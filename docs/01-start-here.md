# Start Here

## Primary Resource

- [GSAP Scroll](https://gsap.com/scroll/)

Use this as your reference for scroll-driven motion patterns.

## What To Do First

1. Choose a progress tracking path:
   - Option A: use GitHub issue template `Project 2 Completion Checklist`
   - Option B: use `docs/02-project-completion-checklist.md` directly
2. Build your project using the structure in [Project Structure Suggestion](04-project-structure-suggestion.md).
3. Track progress in your chosen tracker as you complete required items.
4. Verify your final submission against [Final Turn-In Spec](03-final-turn-in-spec.md).

## Rule Of Focus

- Complete `Required` items first.
- Do not add stretch effects until required items are stable.
